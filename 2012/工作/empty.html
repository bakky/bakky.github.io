String和StringUtil的判断为空区别


1.str.isEmpty();return str==null||str.length()==0

str为null,则抛出空指针异常

2.StringUtils.isEmpty(str)返回true的条件是str==null 或 str.length()==0 null安全

3.StringUtils.isBlank(str)返回true的条件是str==null 或 str.length()==0 或 由空白符(whitespace) 构成

4.StringUtils.trimToNull(str)、StringUtils.trimToEmpty(str) 去掉字符串两端的控制符(control characters, char <= 32) ,如果变为 null 或”“，则返回 null、””
